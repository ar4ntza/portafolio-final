Portafolio #1: Deployment. ☆ *:・ﾟ







Descripción del trabajo:

El deployment de un sitio web significa que está implementando los cambios que ha realizado en su sitio web, normalmente código, desde el control de código fuente hasta un entorno (normalmente desarrollo, ensayo o en vivo) .

Este trabajo tuvo el objetivo de hacer nuestro primer deployment utilizando un acercamiento al entorno virtual usando github pages.
