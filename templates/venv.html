Venv y Flask. (.❛ ᴗ ❛.)



Descripción del trabajo:

El módulo venv proporciona soporte para crear «entornos virtuales» ligeros con sus propios directorios de ubicación, aislados opcionalmente de los directorios de ubicación del sistema.

Este trabajo tuvo el objetivo de conocer y trabajar con múltiples tecnologías como lo fue el framework de flask y un entorno cirtual en python.

Los entornos virtuales de Python ayudan a desacoplar y aislar las instalaciones de Python y los paquetes pip asociados . Esto permite a los usuarios finales instalar y administrar su propio conjunto de paquetes. Link al repositorio ♡
